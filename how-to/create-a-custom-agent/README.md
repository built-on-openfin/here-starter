
# How To Create a custom Agent

## Welcome to the Agent API How To

This how to provides an example of how to create a custom agent that can be used within Enterprise Browser.

You will need to have access to an Enterprise Browser instance and you will need Admin rights so that you can add a search agent.

Instructions on how to configure an agent can be found here: <https://resources.here.io/docs/guide/admins/agents>.

### Examples

| Example                                                      | Description                                                  |
| ------------------------------------------------------------ | ------------------------------------------------------------ |
| [Bloomberg TCAPI Agent](./bloomberg-agent/bbg-tcapi)         | This is a specialized agent that connects to the Bloomberg Terminal and uses TCAPI and GraphQL to send queries directly to it. |
| [Bloomberg TCAPI Client](./bloomberg-agent/bbg-tcapi-client) | This client is a sample illustrating how to use the Bloomberg TCAPI Agent. You can send mnemonics and data directly to the agent and this data will be displayed on the Bloomberg Terminal. |
